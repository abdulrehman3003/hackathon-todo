# Todo Frontend

This is the frontend for the Hackathon II Todo application, built with React and Vite.

## Setup

1.  **Navigate to the frontend directory:**
    ```bash
    cd frontend
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Configure Environment Variables:**
    The application expects `VITE_API_URL` to be set. Create a `.env` file in the `frontend` directory:
    ```
    VITE_API_URL=http://localhost:8000
    ```
    This should match your backend API URL.

4.  **Run the application:**
    ```bash
    npm run dev
    ```
    The frontend will be available at `http://localhost:5173`.
