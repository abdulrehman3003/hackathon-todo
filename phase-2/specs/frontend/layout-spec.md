# Layout & Visual Hierarchy

Top-level:
- App container centered with max-width: 1200px on desktop
- Sidebar (desktop) fixed left; collapsible on mobile
- Main content scrolls independently

Spacing:
- Use CSS variables for spacing (--gap, --space-sm etc)

Typography:
- H1: 2.25rem, bold
- H2: 1.25rem, semibold
- Body: 16px, 1.5 line-height

Shadows:
- Subtle card shadow defined with variable --shadow-sm

Transitions:
- Use transition: transform 160ms ease, opacity 200ms ease for interactive elements
